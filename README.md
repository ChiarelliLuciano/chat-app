## Introducción

Aplicación fullstack de mensajería en tiempo real. 

## Stack

- NextJS, ReactJS, Typescript
- MongoDB, Prisma, Pusher
- Tailwind, HeadlessUI

## Funcionalidades

- Inicio de sesión con Google o Github
- Chats uno a uno
- Chats grupales
- Status activo en tiempo real
- Envío de imágenes
